## Overview

Monorepo of web apps I've built for Hive blockchain in 2019-2020.
The projects are no longer supported or updated.

## Apps

- [Hive Apps Explorer](/hive/hive-apps-explorer/) - a blockchain explorer for Hive dapps.
- [Hive DAO Vue](/hive/hive-dao-vue/) - a governance/proposal system for Hive blockchain.
- [HivedScan](/hive/hivedscan/) - all-in-one blockchain explorer for Hive-based dapps.
